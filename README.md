# Гостевая книга — Serverless веб-приложение
Результат доступен по ссылке: https://d5d6rpnsp29i770d0pfk.kf69zffa.apigw.yandexcloud.net/
Данное приложение представляет собой простую гостевую книгу, реализованную с использованием serverless-контейнеров Yandex Cloud. Пользователи могут оставлять сообщения, которые сохраняются в YDB и отображаются в интерфейсе приложения.

## Архитектура приложения

### Фронтенд

* Фронтенд является статическим веб-приложением, доступным из интернета.
* Развертывание выполнено с использованием Object Storage, а сам доступ осуществляется через API Gateway.
* Приложение доступно по HTTPS через бесплатный служебный домен Yandex Cloud.
* В приложении можно оставить сообщения и посмотреть послания предыдущих посетителей.
* Также в интерфейсе отображается версия фронтенда, бекенда, ID инстанса.

### Бэкенд

* Бэкенд реализован в виде нескольких реплик, развернутых с помощью  Serverless Containers.
* Каждая реплика имеет собственный ID. При многократной перезагрузке на фронте видно, как меняется Instance ID.
* Бэкенд скрыт за API Gateway, благодаря чему каждый запрос может обрабатываться разной репликой бэкенда.

### Хранение данных

* Все данные приложения (сообщения гостевой книги) сохраняются в Serverless YDB.

### Сетевой доступ и безопасность

* Приложение имеет публичный доступ из интернета по HTTPS.
* Доступ к контейнеру осуществляется через API Gateway.

## Автоматизация развертывания

В репозитории веб-приложения присутствуют Bash-скрипты для автоматизации операций развертывания и обновления.
